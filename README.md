# sukoshi-dungeon

Godot Engine 4.5 で作成された、プロシージャル生成されるダンジョンを探索するローグライクゲームです。

## 特徴

*   **プロシージャル生成ダンジョン**: セルオートマトンと部屋接続アルゴリズムを組み合わせた、毎回異なるダンジョン生成。
*   **ターン制ゲームプレイ**: プレイヤーが動けば敵も動く、伝統的なローグライクスタイル。
*   **インベントリシステム**: アイテムを拾い、管理し、使用することができます。
*   **階段**: フロアを降りて深層を目指します。

## 操作方法

| アクション | キー操作 |
| :--- | :--- |
| 移動 | 矢印キー / WASD |
| 待機 | Shift + Space |
| 決定 / インタラクト | Space |
| インベントリを開く | E |
| 戻る / メニュー | Escape |

## 開発環境

*   [Godot Engine 4.5](https://godotengine.org/)

## ディレクトリ構造

*   `addons/`: 使用しているプラグイン (Loggieなど)
*   `assets/`: 画像などのアセットファイル
*   `scenes/`: Godotのシーンファイル (.tscn)
*   `scripts/`: ゲームのロジックを記述したGDScript
    *   `autoload/`: グローバルにアクセス可能なシングルトン (GameManager, SignalBus)
    *   `Entity/`: プレイヤー、敵、アイテムなどのエンティティ定義とコンポーネントシステム
    *   `Game/`: ゲームのメインループやイベントハンドリング
    *   `GUI/`: インベントリやメッセージログなどのUIロジック
    *   `Map/`: ダンジョン生成アルゴリズム (セルオートマトンなど) とマップデータ構造
